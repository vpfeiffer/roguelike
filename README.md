# Roguelike
* Written in Rust
* Nix build script

# Instructions
To run
1. Have NixOS Linux distribution or install on an alternative distribution with [these](https://nixos.wiki/wiki/Nix_Installation_Guide) instructions
2. In the top level directory run
   ```
   nix develop
   ```
3. Run the package with `cargo`.
   ```
   cargo run
   ```
